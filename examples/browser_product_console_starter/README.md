# Browser Product Console Starter Package

This package contains deterministic demonstrative artifacts for the local
read-only Browser Product Console.

Every artifact is classified `DEMONSTRATION_ONLY`. The package is not current
market data, investment advice, a live signal, or an executable portfolio.
It demonstrates A-share, US-equity, and BTC presentation while preserving
mandatory Operator review.

The package includes demonstration-only model governance and factor-governance
projection artifacts so the default Governance workspace shows its complete
read-only state and explicit OBSERVED versus INFERRED field presentation.

The `index.json` file registers every artifact path and SHA-256 digest. The
console fails closed if a registered artifact is missing or modified.
