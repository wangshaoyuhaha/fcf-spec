# AI-CAUSAL-REASONING-CHAIN-APP-1 D1

## Stage

AI-CAUSAL-REASONING-CHAIN-D1

## Purpose

Define the deterministic boundary and anti-hallucination contract for
registered causal reasoning evidence.

This stage does not determine causal truth.

## Allowed inputs

D1 permits future stages to read registered:

- AI context artifacts
- AI evaluation artifacts
- AI challenge artifacts
- market narrative artifacts
- scenario simulation artifacts
- causal claim artifacts
- premise artifacts
- supporting evidence artifacts
- counterevidence artifacts
- alternative explanation artifacts
- correlation rollup artifacts
- validation baseline artifacts
- operator review artifacts

## Allowed outputs

The sidecar may later produce:

- registered causal claim records
- deterministic causal chain records
- chain gap reports
- causal contradiction reports
- alternative explanation reports
- governance review packets
- operator review handoffs

## Anti-hallucination boundary

The sidecar must not:

- create missing causal claims
- create missing evidence
- infer causation from correlation
- convert uncertainty into certainty
- select a causal explanation as true
- assign causal probability
- replace original conclusions
- mutate source artifacts

## Interpretation state

- causal truth: UNDETERMINED
- probability: NOT_ASSIGNED
- winner: NOT_SELECTED
- operator review: REQUIRED
- source artifacts: PRESERVED
- original conclusions: PRESERVED

## Anti-overlap boundary

Existing sidecars remain authoritative for:

- AI context
- AI evaluation
- challenge evidence
- market narrative
- scenario simulation
- correlation rollup
- validation baselines
- operator review

Causal reasoning produces additional governance evidence only.
