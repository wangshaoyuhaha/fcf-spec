# BROWSER-PRODUCT-CONSOLE-OPERATOR-LAUNCH-APP-1 D3

Status: IMPLEMENTED ON SIDECAR BRANCH

## Purpose

D3 turns the existing runtime script into a guided local product entry point.

## Delivered

- no-argument demonstrative starter profile
- `--check` nonblocking startup validation
- optional custom registered root and index
- paired custom argument validation
- exact loopback browser URL validation
- explicit Operator-invoked browser opening
- `--no-browser` support
- A-share, US-equity, and BTC UI rendering acceptance

Running the script remains an explicit Operator action. The server does not
autostart with Windows and never binds outside `127.0.0.1`.
