## Scope

D2 implements registered local JSON loading with:

- artifact identity verification
- SHA-256 verification
- allowed-root containment
- symbolic-link rejection
- schema validation
- correlation_id validation
- observation-window validation
- duplicate record rejection
- read-only source preservation

No external data fetching or network runtime is created.
