## Purpose

Assemble version-locked registered source content into a deterministic
comprehensive report draft.

model, execute a prompt, or replace any upstream conclusion.

## Source Content Rules

Every report statement must come from a registered source payload.

Each content item preserves:

- item identifier
- statement type
- statement text
- conclusion state
- uncertainty state
- risk flags
- reason codes
- evidence references
- counterevidence references
- alternative explanation references

## Assembly Rules

The assembler may only:

- copy registered content
- group registered content
- order registered content
- build source indexes
- expose registered uncertainty
- expose registered risk flags
- expose registered counterevidence
- expose registered alternative explanations

The assembler must not:

- paraphrase source statements
- invent report statements
- invent evidence
- remove risk flags
- remove counterevidence
- remove alternative explanations
- replace source conclusions
- convert uncertainty into certainty
- infer causality from correlation
- select a preferred scenario
- assign probability
- select a winner

## Required Governance States

- causal truth: UNDETERMINED
- probability: NOT_ASSIGNED
- winner: NOT_SELECTED
- operator review: REQUIRED

## Runtime Restrictions

- live model invocation: false
- prompt execution: false
- runtime orchestrator execution: false
- automatic archive execution: false
- trade action generation: false
- real execution: false
