# V2-R1 Factor Contract Foundation App 1 D2

Status: IMPLEMENTED

D2 implements immutable metadata contracts for Factor Definition and Forecast
Target. The contracts validate identity, point-in-time requirements, formula
and parameter versions, fields, sources, history, processing policies, market
regimes, risk, evidence, dependencies, deterministic test vectors, target
maturity, cost, slippage, capacity, labels, abstention, and metrics.

Formula fields are declarations only. This phase performs no technical
indicator calculation, prediction, ranking, or official scoring.

P1-P47 frozen; no P48. The production factor runtime remains not implemented.
No broker, exchange, credential, account, balance, position, wallet, order,
execution, tag, release, or deployment path is added.
