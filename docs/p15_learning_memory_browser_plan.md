# P15-D6 Learning Memory Browser Plan

Status: completed

Purpose:
- provide a read-only local index for learning memory artifacts
- help operator inspect what the learning engine recorded
- keep evidence traceable after release

Readable browser candidates:
- shadow ledger summary
- expert trust score history
- feature audit summary
- anomaly detection summary
- governor weight proposal history
- scenario engine summaries
- patch proposal sandbox summaries
- data quality sentry summaries

Allowed:
- local read-only report
- static JSON summary index
- markdown evidence report
- operator explanation view

Forbidden:
- parameter auto-update
- model auto-deploy
- real exchange API
- API keys
- real orders
- real execution
- real balances or positions
- real money impact
