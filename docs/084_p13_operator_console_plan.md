# P13 Operator Console Plan

Status: planning

Goal:
- Build a local paper-only operator console.
- The console helps the user view project state, validation status, model registry state, paper release status, and safety gates.

Not allowed:
- no real exchange API
- no real brokerage API
- no API keys
- no wallet private keys
- no real orders
- no real execution
- no real balances or positions
- no real money impact
- no bypass of operator review

P13-D1 to P13-D3 target:
- create local UI skeleton
- show current project state
- show paper-only safety boundary
- show validation summary

UI direction:
- local-only web page
- read-only first
- no trading buttons
- no real connection settings
- operator review required
