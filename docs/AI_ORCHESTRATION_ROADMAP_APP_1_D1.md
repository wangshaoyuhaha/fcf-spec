AI-ORCHESTRATION-ROADMAP-D1

orchestration architecture.

D1 may define:

- registered artifact input boundaries
- non-executable roadmap outputs

## Anti-overlap boundary

paper-only local research workflow.

artifacts and governance outputs.

- replace RESEARCH-WORKFLOW-APP-1

- create a runtime AI orchestrator
- modify P1-P47 core
