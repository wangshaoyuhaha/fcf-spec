## Scope

D3 implements registered risk flag inspection, contradiction evidence
inspection, and AI evidence drill-down on top of the D2 evidence graph.

## Delivered

- EvidenceRiskFinding immutable risk evidence
- EvidenceContradictionFinding immutable contradiction evidence
- EvidenceAIDrilldown immutable AI evidence identity
- EvidenceRiskAIDossier deterministic combined evidence view
- explicit risk field allowlist
- ranked watchlist candidate risk flag extraction
- explicit contradiction code allowlist
- registered CONTRADICTS relationship target preservation
- AI Explanation evidence drill-down
- AI Evaluation evidence drill-down
- deterministic model label extraction
- deterministic Prompt version extraction
- deterministic evaluation state extraction
- registered path and content SHA-256 preservation
- deterministic evidence key visibility
- deterministic risk flag counts
- deterministic contradiction code counts
- deterministic AI artifact type counts
- malformed explicit evidence rejection
- graph and read-model correlation validation
- graph and read-model artifact identity validation
- no inference from generic warning or concern fields

## Evidence authority

Only explicit registered fields are inspected.

Risk fields:

- risk_flags
- risk_flag
- risk_codes
- risk_code

Contradiction fields:

- contradiction_codes
- contradiction_code

Generic warning, concern, narrative, or free-text fields are not converted into
risk or contradiction evidence.

AI evidence remains advisory only. The Evidence Audit Explorer does not approve,
promote, activate, learn from, archive, or execute AI output.
